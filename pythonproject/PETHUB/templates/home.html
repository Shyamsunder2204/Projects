Home


In proposed system we are providing an online portal. The main objective of a pet care application is to provide essential information about the pet animals in virtual way. The pet owner can easily access the information about the pet animals through this application. It is an easy way as the pet owner can check the information in seconds by using this application. It reduces the cost and efforts of the pet owner as they do not need to visit pet care center. we can gain more relevant information by browsing the application rather than visiting the pet center.

In existing system, pet care system all the procedures are done manually, the main drawback of system is time consuming. If we need any information about the pet, we need to visit the pet care center as we are unaware of their pet needs. If the pet is suffering from any disease or it is unwell, we need to visit the pet hospital. we will feed the food to the pet which might harm the pet as we are unaware of their food habits. It is slightly expensive to visit the pet care center every time. To overcome the above problems, we need an online application which solves all the queries about the pet.
